LTC2983/LTC2984/LTC2986/LTM2985 Multi-Sensor Digital Temperature Measurement System Driver¶

Supported Devices¶

  • LTC2983

  • LTC2984

  • LTC2986

  • LTM2985

Evaluation Boards¶

  • DC2296A

  • DC2296A-KIT

Description¶

This is a Linux industrial I/O (IIO) subsystem driver, targeting temperature sensors. The industrial I/O subsystem provides a unified framework for drivers for many different types of converters and sensors using a number of different physical interfaces (i2c, spi, etc). See IIO for more information.

Example SPI device initialization¶

The devices can only be instantiated via device tree, along with all the sensors attached to it.

Adding Linux driver support¶

Configure kernel with “make menuconfig” (alternatively use “make xconfig” or “make qconfig”). <WRAP round help> The LTC2983 driver depends CONFIG_SPI. </WRAP>
